# Astronaut-voice-distortion-Pro.

A test script for astronaut voice distortion applications, as well as other relevant commercial use cases 
requiring the evaluation of specialized audio effects prior to release, is presented in this repository. 
Improved usability is achieved through the implementation of slider-based controls, thereby eliminating the 
need to directly modify parameter values within the underlying DSP algorithm at the source code level. 
This approach enables faster and more visually intuitive feedback, facilitating the efficient adjustment of 
distortion levels in accordance with the specific requirements of diverse use cases.
